Technologies & Compétences

01

Laravel

Laravel est un framework PHP robuste pour le développement d'applications web. Il facilite la gestion des bases de données, l'authentification et l'architecture MVC.

02

ReactJs / NextJs

ReactJs est une bibliothèque JavaScript pour la création d'interfaces utilisateurs interactives. NextJs, basé sur React, permet le rendu côté serveur et l'optimisation des performances pour les applications web.

03

Flutter

Flutter est un framework de développement d'applications mobiles multiplateformes. Il permet de créer des applications natives performantes pour iOS et Android avec un seul code source.

04

Kotlin

Kotlin est un langage moderne et concis pour le développement d'applications Android. Il est totalement compatible avec Java et facilite la création d'applications fiables et rapides.




05

Java(FX)

JavaFX est un framework Java pour la création d'interfaces graphiques modernes et interactives. Il permet de développer des applications desktop avec des éléments visuels riches.

06

UI/UX Design

Le design UI/UX se concentre sur la création d'interfaces utilisateurs esthétiques et intuitives. Il vise à améliorer l'expérience utilisateur à travers une conception simple, fonctionnelle et agréable.

07

C# & Xamarin Forms

Xamarin Forms est un framework pour le développement d'applications mobiles multiplateformes avec C#. Il permet de créer des applications natives pour iOS, Android et Windows à partir d'une base de code partagée.

08

Gestion de projet

La gestion de projet consiste à planifier, organiser et superviser les projets de développement. Elle garantit le respect des délais, du budget et des exigences des clients.
